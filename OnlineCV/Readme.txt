
This project OnlineCV has been developed as a part of Tigerspike recruitment process.

Summary of development :

The OnlineCV project is developed on Visual studio express for web 2013 and mainly used below technologies.

Aspnet.MVC5
Bootstrap
AngularJS
How it works !

All the CV contents are in /Contents/mycv.json. 
The javascript /Scripts/onlinecv-angular.js will read the mycv.json and load it into angular scope.

Then angular scope has been bind on html tags for the .cshtml pages as required. 
In this project i have used only 1 angular app and 1 controller. 

Contact form will send email to arunonlinecv@gmail.com with form contents. Gmail password for this account is aruncv1234

Boostrap has been used to arrange the layouts, 
Javascript/JQuery related to this controllers are written under /Scripts/Controllers/
